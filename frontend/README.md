# Frontends

## Setup
Run:

```shell
npm install
npm run build
```

Then the webpack server:
```shell
npx webpack serve
```

Next go to: http://0.0.0.0:8081/?crosswordId=b6c5f2d51d374129a1b2e5b9b64d9161
